Link Browser

Prototype Build

A lightweight JavaFX-based web browser built for simplicity, performance, and a non-Google Internet experience.

The browser is inspired by popular browsers such as Chrome, Opera, and Firefox. However, its primary focus is to provide a browsing experience using non-Google or big-company controlled search engines.

It is written in Java using OpenJDK 23, which allows cross-platform compatibility and a modern development environment. Our mission is to prioritize user privacy and control over their browsing experience, without relying on the large corporations that dominate the web today.

The build date reflects the date when the project was last compiled and updated with new features.
